# Animals and Nature Messages

REST API for Animals and Messages administrator dashboard where administrators could view user feedback.  This is a Spring Boot REST API that provides CRUD operations for emails pertaining to the Animals and Nature message board.  AWS SNS and AWS Lambda were used to receive email messages and send the messages to the API.  Authorization was built in using OAuth 2.0.  This API is currently hosted on Render.  You would need an account to access the API.  You could view the GitHub repository for the Animals and Nature message board via this [link](https://github.com/starrv/animals-and-nature).  NOTE:  I am currently using the free plan so startup could take several minutes.
